Game of War

Live Demo

The "Game of War" project is a browser-based game that simulates the classic card game War. The game allows a user to play against a computer opponent and tracks the number of rounds played, the number of cards drawn, and the number of wins for each player. The project utilizes the Deck of Cards API to shuffle and draw cards, and the winner of each round is determined by comparing the values of the cards drawn.

The interface of the game is designed with HTML and CSS, and the game logic is written in JavaScript. The user can start a new game by clicking the "New Deck" button, which shuffles the deck and enables the "Draw Cards" button. When the user clicks "Draw Cards," two cards are drawn from the deck and displayed on the screen. The winner of the round is determined by comparing the values of the cards, and the winner's score is updated accordingly.



"Game of War" is an advanced front-end project that demonstrates the developer's proficiency in utilizing third-party APIs, asynchronous functions, and DOM manipulation to create an interactive and engaging user interface. The project's implementation of a fireworks animation to celebrate the game's winner showcases the developer's creativity and attention to detail, adding a visually appealing touch to the project. The project's overall technical complexity and aesthetic quality make it a notable achievement in front-end development.
