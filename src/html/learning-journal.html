Learning Journal

Live Demo

I built this Learning Journal from scratch using HTML, CSS, CSS-Flexbox, and JavaScript. To ensure that all pages are responsive and work well on smaller screens, I used a mobile-first design approach. The project includes custom images and content about my learning experiences. I also added new pages, a responsive navigation menu with a hamburger menu, and a "View More" feature to display other recent posts.

I paid close attention to detail, as seen in the Figma design screenshots, to create a visually appealing and user-friendly website. I used relative units, media queries, and the viewport meta tag to ensure that the website is responsive and accessible to all users. As a stretch goal, I added custom images, new pages, and a feature to display other recent posts.



Through this project, I was able to highlight my proficiency in HTML, CSS, and JavaScript, as well as my ability to design and develop a responsive website. I believe that my attention to detail and the mobile-first approach I used in this project demonstrate my commitment to creating an exceptional user experience. Overall, I am proud of what I accomplished with this project and look forward to further improving my skills in web development.
