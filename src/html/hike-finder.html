Hike Finder

Live Demo

The HikeFinder website in my web developer portfolio serves as a testament to my expertise in building responsive and engaging websites. I designed the homepage and explore page using HTML, CSS, CSS-Flexbox, and JavaScript, prioritizing mobile-first design principles and incorporating media queries to ensure optimal viewing on all devices. Furthermore, I went beyond the core requirements of the project by adding a journal page that allows users to document completed hikes. Overall, HikeFinder is a strong example of my ability to create high-quality, responsive websites that exceed client expectations.

One of the standout features of HikeFinder is the embedded Google map on the explore page, which enables users to view hiking trails and plan their next adventure. Additionally, as a stretch goal I designed and built the sign-up and login form pages, which are essential for users to create an account and access the full functionality of the website. The project also includes a responsive nav, which ensures that users can easily navigate through the site, regardless of their device.



Overall, HikeFinder demonstrates my ability to meet project requirements, while also going above and beyond to achieve stretch goals. In addition to building the home and explore pages from scratch, I also ensured that each page was responsive, used flexbox and/or grid, and implemented media queries. By showcasing this project in my portfolio, I hope to demonstrate my skills as a web developer and my commitment to delivering high-quality projects that meet client needs.
