Employee Directory

Live Demo

The Employee Directory project showcases a Figma design that the I used to guide the development process. This demonstrates the importance of designing before development, ensuring that the end product meets the client's needs and is visually appealing. The live demo of the project is available on Netlify, making it easy to view the final product and interact with it.

The Employee Directory project is a great demonstration of using several programming languages to build a functional and visually appealing web application. The project was built from scratch using HTML, CSS, and JavaScript, and it utilizes CSS flexbox and grid, as well as JavaScript classes. The web application displays employee information such as image, name, title, bio, and social info. Additionally, the project meets the requirement of filtering data to display only people on the selected team.



One of the impressive aspects of this project is the addition of a search filter that allows users to search for team members by name. This is a stretch goal that I added, making it more interactive and user-friendly. Furthermore, this project demonstrates my proficiency in using various programming languages to build a functional web application.
