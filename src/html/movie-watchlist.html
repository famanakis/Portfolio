Movie Watchlist

Live Demo

As a developer, I always enjoy working on projects that allow me to showcase my skills and creativity. The Movie Watchlist project was a fun solo project that gave me an opportunity to integrate Web APIs, write JavaScript code, and use CSS to create a beautiful and responsive web application. I approached this project with the goal of building a clean and easy-to-use interface that would allow users to search for their favorite movies and add them to their watchlist.

To build this project, I used a combination of technologies such as JavaScript, HTML, and CSS to create two pages - index.html and watchlist.html. The index.html page serves as the search page and allows users to search for movies using the Open Movie Database (OMDB) API. The watchlist.html page, on the other hand, loads and displays data from local storage, allowing users to view the movies they have added to their watchlist. I also used CSS-Flexbox to ensure that the application is responsive across different devices.



In addition to delivering a functional and visually appealing Movie Watchlist project that met all the requirements, I am particularly pleased with how I incorporated local storage to enhance the user experience. By utilizing local storage, users are able to save their movie selections and easily retrieve them upon returning to the website. This feature not only improves usability but also demonstrates my proficiency in front-end development. Overall, I am proud of the work I accomplished on this project and the skills I showcased.
