Quizzical

Live Demo

I am excited to present my Quizzical project on my developer portfolio website. This solo project utilized React Framework, JavaScript, CSS-Flexbox, HTML, and CSS to create a two-screen quiz app. The app pulls five questions from the OTDB API and tallies the user's correct answers after they click "Check answers." Additionally, I focused on styling and polishing the app to create a visually appealing user interface.

During the development of this project, I challenged myself to make the app responsive by using a mobile-first design approach. This meant that the app was optimized for mobile devices and adjusted to larger screens without sacrificing functionality or design. Additionally, I incorporated stretch goals, allowing users to choose the topic for their quiz questions and the level of difficulty. These additional features showcase my commitment to creating an engaging user experience and attention to detail.



Overall, the Quizzical project is a testament to my abilities as a developer. It showcases my skills in using React Framework and other programming languages while creating a user-friendly and visually appealing application. Additionally, the incorporation of stretch goals demonstrates my commitment to continuously improving and iterating on my projects. I am proud to have this project as a part of my portfolio and look forward to showcasing it to potential employers and clients.
